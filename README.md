# simple-ortools-vrp

Simple prototype use [OR-Tools](https://developers.google.com/optimization) for solve vehicle routing problem and plot result to display

### Installation

```
install python3
install ortools via python3 `python3 -m pip install ortools`
```

### Run

`python3 ./main.py`
